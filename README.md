# Galaxy Gazer

## Purpose
Galaxy Gazer is an interactive web application built to demonstrate core front-end development skills, including public API integration, UI design, and JavaScript logic. The project uses the NASA APOD API to fetch and display astronomical data. It features a unique dark-themed interface and utilizes JavaScript array methods to allow users to filter, sort, and search through collections of space photography.

## API Integration
* **API:** NASA Astronomy Picture of the Day (APOD) API
* **Base URL:** `https://api.nasa.gov/`


## Planned Features
* **Date Explorer:** Users can select a specific date to fetch the APOD for that day.
* **Cosmic Gallery:** Fetches a collection of historical APOD data to browse, with add to favourites functionality and users can read about the image by clicking on "Read More" button.

## Technologies Used
* **HTML5:** For the structural foundation.
* **CSS3:** For styling, specifically focusing on a dark space theme with glowing elements and high-resolution backgrounds. 
* **JavaScript (ES6):** For DOM manipulation, API fetching, and array higher-order functions.
